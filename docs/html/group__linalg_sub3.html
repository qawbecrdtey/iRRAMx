approximately reduce a square matrix to Hessenberg matrix

For a square matrix M, compute a Hessenberg matrix H. It holds that the eigenvalues of H are perturbed from the eigenvalues of M by at most 2^p

Parameters
Ma square matrix
pnegative integer
Returns
a Hessenberg matrix whose eigenvalues approximates the eigenvalues of M
